# 🎙️ Voice Recorder

A simple and lightweight Voice Recorder project that allows users to record, play, and manage audio recordings easily.


---

## ✨ Features

🎤 Record voice/audio instantly

▶️ Play recorded audio

⏸️ Pause and resume recording

💾 Save recordings locally

📂 Easy file management

📱 Simple and clean interface

⚡ Fast and lightweight



---

## 🚀 Installation

### Clone the Repository
```
git clone https://github.com/surya404root/Voice-Recorder.git
```
### Open Project
```
cd Voice-Recorder
```

---

## ▶️ Usage

1. Open the application.


2. Press the Record button.


3. Speak into the microphone.


4. Stop recording when finished.


5. Play or save your recording.
